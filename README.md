# Scann Pro: 现代化单细胞检索与分析系统

Scann Pro 是一个专为生物信息学设计的全栈单细胞向量检索系统。它结合了高效的近似最近邻（ANN）算法与响应式的现代化 Web 前端，支持海量细胞数据的毫秒级比对与性能评测。

## ? 核心特性

### 1. 极速检索体验
*   **多模式搜索**：支持通过 `Cell ID` 在库内搜索相似细胞，或直接输入 `Vector` 向量进行跨库比对。
*   **智能过滤**：支持基于元数据（Metadata）的精准过滤，可通过下拉菜单动态选择 `cell_type`, `disease` 等字段进行约束检索。
*   **排除自身逻辑**：检索结果自动剔除查询对象本身，确保结果的生物学分析意义。

### 2. 专业数据管理
*   **一键构建索引**：集成 HNSW (ANN) 与 Exact (暴力) 两种搜索算法，支持在线异步构建。
*   **数据上传**：支持 `.h5ad` 标准单细胞格式文件的一键上传与元数据自动解析。
*   **动态展示**：自动提取并展示细胞的所有 obs 标签，支持标签云式的元数据可视化展示。

### 3. 深度的性能标测 (Benchmarking)
*   **加速比分析**：自动对比 ANN 与精确检索的耗时，通过图表直观展示 Speedup 倍数。
*   **准确率验证**：计算 Recall@K 指标，确保加速的同时维持极高的科学准确性。

### 4. 安全访问控制 (RBAC)
*   **五级权限体系**：从 `Visitor` 到 `SysAdmin` 的严格权限隔离，功能菜单根据角色动态展示。

## ? 快速启动

### 1. 环境准备
确保已安装 Python 3.8+，然后执行：
```bash
pip install -r requirements.txt
```

### 2. 初始化系统
此操作将清除旧数据，初始化数据库结构并自动扫描 `data/uploads/` 目录下的演示数据，同时创建预设账号：
```bash
python reset_all.py
```

### 3. 运行项目
启动 Flask 服务器：
```bash
python run.py
```
访问地址：`http://127.0.0.1:5000`

## ?? 技术架构
*   **后端**: Flask + SQLAlchemy + Scanpy (数据处理) + Faiss/HNSWlib (向量引擎)
*   **前端**: React 18 (Cdn Buildless) + TailwindCSS (响应式 UI) + Axios
*   **数据库**: SQLite3

## ? 系统开发建议
1.  **索引预构建**：在使用检索前，请务必先在【数据与索引】页面点击“构建 HNSW”按钮。
2.  **元数据匹配**：检索时的过滤值区分大小写，建议先通过“随机测试”观察结果中的标签写法。
